--- title: About this theme layout: page ---

Minimal and responsive Jekyll theme with a clear font, custom post header, SVG icons, Disqus and Google Analytics.

Features

  • Responsive Design
  • Minified CSS
  • Custom Post Header
  • SVG Icons
  • Syntax Highlighting
  • Google Font: Martel
  • Google Analytics
  • Disqus

Hope you enjoy it!

Credits

The whole theme is built on top of Tachyons, a small and easy to use css framework. Social svg icons are from Geomicons.
